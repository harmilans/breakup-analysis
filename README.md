# Talking Stage Autopsy

Standalone, free, static frontend. No API key, backend, billing, or screenshot upload is required.

## Files

- `index.html` - the full app UI and local vent-text analysis.

## Run Locally

Open `index.html` directly in a browser.

## Deploy

Deploy the `talking-stage-autopsy` folder to Vercel as a static site.

No environment variables are needed. The app analyzes vent text locally in the browser using keyword scoring and does not upload or save submissions.
